* Introduction
  This document illustrates the installation of [[http://alloy.mit.edu/alloy/download.html][alloy]].

* Installation Steps
  
  1. Install Open JDK
     #+BEGIN_EXAMPLE
     sudo apt-get install openjdk-8-jre
     #+END_EXAMPLE

  2. Download alloy
     #+BEGIN_EXAMPLE
     wget http://alloy.mit.edu/alloy/downloads/alloy4.2.jar
     #+END_EXAMPLE

* Run Alloy
  #+BEGIN_EXAMPLE
  java -jar alloy4.2.jar
  #+END_EXAMPLE
